# 《边狱巴士》通行证追赶组合包策划案（优化版）

> 本文为个人学习与求职作品展示使用。文中涉及《Limbus Company / 边狱巴士》的游戏名称、系统概念与部分资源名，其版权归 Project Moon 所有。本文仅用于商业化系统与活动方案设计分析，不作商业用途。

## 0. 一页概述

**方案名称**：赛季末通行证追赶组合包  
**方案类型**：赛季末商业化礼包 + 新手/回流追赶机制  
**适用阶段**：赛季结束前 3-4 周  
**目标用户**：赛季末未购买付费通行证的新玩家、回流玩家、低等级玩家  
**核心目标**：降低“现在买通行证也拿不完”的心理阻力，提高赛季末通行证购买率与新玩家留存。

### 核心结论

1. 追赶组合包的本质不是单纯卖资源，而是解决赛季末玩家的购买犹豫：剩余时间不足、奖励拿不满、买了不划算。
2. 组合包只面向“未购买当前赛季付费通行证”的低等级玩家，内容为“当前赛季付费通行证 + 自动等级直升 + 体力资源 + 核心养成材料”。
3. 已购买付费通行证的玩家不出售等级直升，只可购买“养成补给包”。该规则用于避免付费玩家通过低价直升提前进入 120 级后的无限碎片箱阶段。
4. 组合包价格必须高于单独通行证价格，建议为通行证价格的 1.3-1.5 倍。晚买玩家支付的是“通行证本体 + 时间追赶服务”，避免形成“等赛季末再买更赚”的预期。
5. 等级直升购买后立即自动生效，不以道具形式进入背包，避免囤券、未使用、赛季结束转换等客服纠纷。
6. 直升上限建议为 Lv.80。超过上限的直升等级直接作废，不做补偿，但 UI 必须在购买前明确展示“本次实际提升等级”。
7. 礼包不补发大量狂气、赛季纪念装饰或 120 级后的无限溢出收益，避免伤害早期玩家的公平感。

---

## 1. 背景问题

赛季通行证天然依赖时间推进。对赛季初或赛季中购买的玩家来说，通行证价值清晰：持续完成日常、周常和镜牢，就能稳定获得免费轨与付费轨奖励。

但赛季末新玩家或回流玩家会面临三个问题：

| 问题 | 玩家感受 | 对产品的影响 |
| --- | --- | --- |
| 剩余时间不足 | “现在买也拿不完” | 赛季末通行证购买率下降 |
| 等级差距过大 | “我已经错过太多奖励” | 新玩家追赶压力上升 |
| 初期战力不足 | “买了也刷不动/刷不快” | 新玩家活跃与留存下降 |

因此，赛季末需要一个“补入口”，让晚入坑玩家仍然有理由购买通行证，并进入日常、镜牢、碎片箱循环。

但该方案必须同时保护两类玩家的心理：

| 玩家 | 需要保护的感受 |
| --- | --- |
| 早期购买玩家 | 早买早享受，不能被赛季末低价礼包背刺 |
| 重度玩家 | 长期肝出的通行证等级和无限箱进度不能被低价直升轻易追平 |

---

## 2. 设计目标

### 2.1 用户目标

- 降低赛季末购买通行证的心理压力。
- 让新玩家购买后立即获得一批可领取奖励和基础养成资源。
- 帮助玩家追到关键奖励段，例如 Lv.60 或 Lv.80，而不是从 Lv.1 开始硬肝。
- 引导玩家继续参与日常、周常和镜牢，而不是买完即走。

### 2.2 产品目标

| 目标 | 说明 |
| --- | --- |
| 提高赛季末通行证购买率 | 让低等级未购玩家仍有购买理由 |
| 保护赛季中期销量 | 组合包提价，避免玩家等待赛季末再买 |
| 提升新玩家 7 日 / 14 日留存 | 通过资源补给降低早期卡点 |
| 增加镜牢参与 | 体力资源和追赶任务引导玩家继续刷取通行证经验 |
| 保持老玩家公平感 | 不给已购玩家卖等级，不补无限溢出收益 |
| 控制资源经济 | 礼包资源以养成追赶为主，不大量投放通用抽卡货币 |

---

## 3. 设计原则

### 3.1 追赶但不补满

组合包帮助玩家追到关键奖励段，例如 Lv.60 或 Lv.80，但不直接补到 Lv.120，更不补偿 Lv.120 后的无限碎片箱收益。

```text
合理追赶：补核心养成与中段奖励
不合理补偿：补满全部通行证、补无限溢出箱子
```

### 3.2 等级直升只绑定未购通行证组合包

等级直升只作为“未购玩家补票”的一部分存在，不对已购买付费通行证的玩家单独出售。

原因：已购玩家如果购买等级直升，会加速进入 Lv.120 后的无限碎片箱阶段，等于用付费直接购买未来溢出箱进度，容易伤害重度玩家的时间投入感。

### 3.3 晚买需要支付时间溢价

组合包价格高于单通行证价格。玩家赛季初购买通行证，获得的是完整赛季时间；玩家赛季末购买组合包，获得的是缩短追赶时间的服务，因此需要额外付费。

```text
早买：通行证原价 + 完整赛季时间
晚买：通行证原价 + 时间追赶溢价
```

### 3.4 减少错过感，不制造强焦虑

界面文案应强调“帮助追赶当前赛季进度”，而不是用强刺激文案制造恐惧，例如“错过就没了”“最后机会”。

### 3.5 购买后立即生效

等级直升不以道具形式进入背包，而是在购买成功后立即自动应用。

这样可以避免：

- 玩家忘记使用直升券。
- 赛季结束后直升券转换争议。
- 玩家囤券等待更高价值转换。
- 客服处理未使用道具纠纷。

## 4. 上架条件

### 4.1 时间条件

| 条件 | 规则 |
| --- | --- |
| 开始时间 | 当前赛季结束前 28 天开启 |
| 结束时间 | 当前赛季结束时关闭 |
| 展示周期 | 最后 4 周全程展示，但根据剩余天数调整推荐文案 |

### 4.2 玩家条件

追赶组合包只对符合条件的玩家展示。

| 条件 | 规则 |
| --- | --- |
| 通行证状态 | 未购买当前赛季付费通行证 |
| 通行证等级 | 当前等级低于 Lv.80 |
| 玩家类型 | 新玩家、回流玩家、低等级活跃玩家均可 |
| 购买次数 | 每赛季限购 1 次 |

### 4.3 已购买通行证玩家处理

已购买当前赛季付费通行证的玩家不展示“通行证追赶组合包”，也不出售等级直升。

可展示一个独立的“赛季养成补给包”：

| 内容 | 是否包含 |
| --- | --- |
| 等级直升 | 不包含 |
| 通行证经验 | 不包含 |
| 碎片箱 | 不包含或极少量随机箱，首版建议不包含 |
| 体力资源 | 包含 |
| 经验券 | 包含 |
| 纺锤 | 包含 |
| 普通养成材料 | 包含 |

该补给包的定位是“养成补充”，不是“通行证进度追赶”。

## 5. 礼包内容设计

### 5.1 标准组合包

**礼包名称**：赛季末通行证追赶组合包  
**购买限制**：每赛季限购 1 次  
**展示对象**：未购买当前赛季付费通行证，且通行证等级低于 Lv.80 的玩家  
**推荐价格**：当前赛季付费通行证价格 × 1.3-1.5

| 内容 | 数量 | 作用 |
| --- | ---: | --- |
| 当前赛季付费通行证 | 1 | 解锁当前赛季付费轨 |
| 自动等级直升 | 20 级 | 降低追赶压力，最多提升至 Lv.80 |
| 体力盒 / 体力资源 | 8-10 个 | 引导继续参与镜牢与日常 |
| 高级经验券 | 30-40 个 | 帮助新玩家快速提升核心人格等级 |
| 纺锤 | 80-120 个 | 支撑人格同步/养成突破 |
| 通用养成材料包 | 1 组 | 补齐早期养成短板 |

### 5.2 付费轨回溯规则

购买组合包后，系统按以下顺序处理：

```text
1. 解锁当前赛季付费通行证
2. 读取玩家当前通行证等级
3. 自动提升指定等级，但最高不超过 Lv.80
4. 一次性开放 1 级至购买后等级的所有付费轨奖励
5. 弹出本次新增可领取奖励汇总
```

示例：

| 当前等级 | 购买后等级 | 玩家可领取内容 |
| ---: | ---: | --- |
| Lv.10 | Lv.30 | 付费轨 Lv.1-Lv.30 奖励 + 直升新增免费轨奖励 |
| Lv.45 | Lv.65 | 付费轨 Lv.1-Lv.65 奖励 + 直升新增免费轨奖励 |
| Lv.70 | Lv.80 | 付费轨 Lv.1-Lv.80 奖励 + 直升新增免费轨奖励 |

UI 必须高亮展示：

```text
购买后预计可领取：付费轨奖励 X 项、碎片箱 X 个、EGO/核心奖励 X 个、养成材料 X 组
```

这是组合包最核心的正反馈点。

### 5.3 直升上限与超出处理

规则：

```text
等级直升最多提升至 Lv.80，超出部分不返还、不转换。
```

示例：

| 当前等级 | 标准直升 | 实际提升 | 结果 |
| ---: | ---: | ---: | --- |
| Lv.10 | +20 | +20 | 到 Lv.30 |
| Lv.45 | +20 | +20 | 到 Lv.65 |
| Lv.70 | +20 | +10 | 到 Lv.80，剩余 10 级作废 |
| Lv.80 | 不可购买 | 0 | 不展示礼包 |

购买前必须展示动态提示：

```text
当前等级 Lv.70，本礼包实际可提升 10 级，最高提升至 Lv.80，超出部分不返还。
```

这样让接近 Lv.80 的玩家自行判断是否购买，减少“同价不同收益”的投诉。

### 5.4 为什么不直接赠送大量碎片箱

通行证本身已经通过等级奖励提供碎片箱。如果礼包额外大量赠送自选碎片箱，可能直接冲击通行证的长期目标与早期玩家公平感。

建议：

- 通过等级直升让玩家获得通行证内已有碎片箱。
- 礼包本体不额外赠送大量自选碎片箱。
- 首版不直接赠送自选碎片箱。

## 6. 价格策略

### 6.1 价格公式

```text
追赶组合包价格 = 当前赛季付费通行证价格 + 时间追赶溢价
```

建议区间：

```text
追赶组合包价格 = 通行证价格 × 1.3-1.5
```

设计目的：

| 目的 | 说明 |
| --- | --- |
| 保护早买玩家 | 早买玩家以原价获得完整赛季时间 |
| 防止中期销量下滑 | 玩家不会因为赛季末礼包更赚而故意等待 |
| 解释直升价值 | 多出的价格对应等级直升与时间差 |
| 降低背刺感 | 晚买玩家不是低价补票，而是付费追赶 |

### 6.2 已购玩家补给包价格

已购通行证玩家的补给包不包含等级和通行证经验，因此价格应低于追赶组合包，并且只按体力与养成材料定价。

```text
赛季养成补给包价格 = 体力资源 + 养成材料价格
```

该礼包不能出现“等级直升”“通行证经验”“快速到 120 级”等文案。

## 7. 购买与领取流程

### 7.1 入口位置

| 入口 | 展示逻辑 |
| --- | --- |
| 通行证界面顶部 Banner | 主要入口，展示“赛季追赶” |
| 商城限时页签 | 次要入口，方便主动购买 |
| 镜牢结算后弱提示 | 仅对低等级且未购通行证玩家展示 |
| 登录弹窗 | 每周最多展示 1 次，避免打扰 |

### 7.2 玩家流程

```text
玩家进入通行证界面
-> 系统判断是否符合追赶组合包展示条件
-> 展示追赶组合包 Banner
-> 玩家点击查看详情
-> 页面展示当前等级、购买后等级、实际提升等级、可领取奖励数量
-> 玩家确认购买
-> 系统解锁当前赛季付费通行证
-> 自动应用等级直升，最高至 Lv.80
-> 发放体力资源与养成材料
-> 弹出本次新增可领取通行证奖励汇总
-> 引导玩家前往镜牢/日常继续获取通行证经验
```

### 7.3 奖励领取规则

购买后奖励分为三部分：

| 奖励来源 | 发放方式 |
| --- | --- |
| 付费通行证 1 级至购买后等级奖励 | 立即解锁，可在通行证界面领取 |
| 等级直升带来的新增免费轨奖励 | 立即解锁，可在通行证界面领取 |
| 礼包额外资源 | 直接进入背包；背包满则进入邮箱 |

## 8. UI 展示需求

### 8.1 Banner 文案

推荐文案：

```text
赛季追赶开启
解锁付费通行证，获得等级助力与养成补给，继续推进本赛季奖励。
```

避免文案：

```text
最后机会！错过无法挽回！
```

原因：后者焦虑感过强，容易被理解为强逼付费。

### 8.2 详情页信息结构

详情页必须清楚展示：

| 模块 | 内容 |
| --- | --- |
| 当前状态 | 当前通行证等级、剩余赛季天数 |
| 购买后变化 | 解锁付费轨、实际提升等级、购买后等级 |
| 可领取奖励 | 付费轨奖励数量、碎片箱数量、EGO/核心奖励数量 |
| 礼包内容 | 通行证、等级、体力、养成材料 |
| 预计进度 | 继续完成日常/周常/追赶任务后预计可到达等级 |
| 重要提示 | 最高提升至 Lv.80，超出等级不返还 |

### 8.3 预计进度展示

预计进度必须把追赶任务作为变量，而不是只计算礼包本身。

示例：

```text
当前等级：Lv.28
购买后等级：Lv.48
剩余赛季：21 天
完成每日/周常后预计可达：Lv.72
完成全部追赶任务后预计可达：Lv.80+
```

该模块的作用是让玩家知道礼包不是买完即结束，而是买后继续玩仍然有明确目标。

## 9. 追赶任务联动

为了避免礼包变成一次性资源售卖，建议同步开放轻量“赛季末追赶任务”。

### 9.1 开放对象

追赶任务不应完全锁在付费后，否则容易被认为“先制造落后，再卖解决方案”。

| 玩家 | 是否开放 |
| --- | --- |
| 未购通行证低等级玩家 | 开放 |
| 已购通行证但等级低玩家 | 开放 |
| 高等级玩家 | 不开放或只显示普通周常 |

### 9.2 任务内容

| 任务 | 奖励 | 目的 |
| --- | --- | --- |
| 完成 3 次日常资源关 | 固定通行证经验 + 经验券 | 引导基础养成 |
| 完成 1 次镜牢 | 固定通行证经验 + 体力资源 | 引导核心循环 |
| 提升 1 名人格等级 | 固定通行证经验 + 纺锤/经验券 | 帮助形成战斗力 |
| 累计登录 3 天 | 固定通行证经验 | 提高短期留存 |
| 完成 5 个日常任务 | 固定通行证经验 | 建立日常习惯 |

### 9.3 奖励口径

追赶任务奖励使用固定数值的“赛季通行证经验”，不使用百分比提升。

原因：

- 方便计算预计可达等级。
- 避免不同等级玩家获得差异过大。
- 便于控制总投放。

## 10. 资源投放控制

### 10.1 不建议投放内容

| 内容 | 原因 |
| --- | --- |
| 大量狂气 | 会冲击抽卡资源经济，引发早买玩家不满 |
| 大量自选碎片箱 | 会压缩通行证长期刷取目标 |
| 赛季纪念装饰 | 应保留早期参与玩家的纪念价值 |
| 120 级后溢出箱子补偿 | 会直接伤害重度玩家时间投入价值 |
| 已购玩家等级直升 | 会加速进入无限箱阶段，引发公平争议 |

### 10.2 推荐投放内容

| 内容 | 原因 |
| --- | --- |
| 通行证等级 | 只用于未购组合包，解决赛季末追赶问题 |
| 体力资源 | 引导继续参与镜牢和日常 |
| 经验券 | 帮助新玩家快速提升战力 |
| 纺锤 | 支撑人格同步与养成 |
| 普通养成材料 | 降低早期卡点 |

### 10.3 资源上限建议

| 资源 | 上限建议 |
| --- | --- |
| 等级直升 | 首版 20 级，最高提升至 Lv.80 |
| 体力资源 | 支撑 3-5 天中度游玩 |
| 经验券/纺锤 | 支撑 1-2 名核心人格成型 |
| 自选碎片箱 | 首版不直接赠送 |

---

## 11. 异常与边界条件

| 情况 | 处理规则 |
| --- | --- |
| 玩家已购买付费通行证 | 不展示组合包，只展示不含等级的养成补给包 |
| 玩家等级高于或等于 Lv.80 | 不展示追赶组合包 |
| 当前等级 + 直升等级超过 Lv.80 | 只提升至 Lv.80，超出部分作废，不返还 |
| 购买成功后 | 等级直升立即自动生效，不发放直升券道具 |
| 赛季结束前未领取奖励 | 按游戏原有通行证未领取规则处理，必要时邮件提醒 |
| 背包满 | 礼包资源进入邮箱，保留 7-14 天 |
| 网络中断 | 以服务器订单状态为准，未发放则补发 |
| 重复购买 | 每赛季限购 1 次，服务器限制购买次数 |
| 退款 | 回收未消耗资源；已消耗部分按项目规则处理 |
| 赛季延期 | 礼包展示时间随赛季结束时间同步延长，并按补偿规则处理 |

### 11.1 赛季延期处理

如果玩家在赛季结束前购买追赶组合包后，官方宣布赛季延期，可能产生“我本可以自己肝，不必买直升”的背刺感。

处理建议：

- 礼包说明页增加提示：赛季结束时间以官方最新公告为准，礼包内容基于当前规划进度设计。
- 若购买后发生赛季延期，可向已购买追赶组合包的玩家补发少量体力资源作为安抚。
- 不返还等级直升价值，避免规则复杂化。

## 13. A/B 测试方案

原方案中“更多体力 vs 更多材料”的差异较弱，不容易测出核心问题。更适合测试“代练式直升”与“门票式养成”的差异。

| 版本 | 内容方向 | 设计假设 |
| --- | --- | --- |
| A 版：加速版 | 通行证 + 15 级直升 + 少量体力 | 休闲玩家更想快速拿到通行证奖励 |
| B 版：养成版 | 通行证 + 10 级直升 + 更多体力 + 更多纺锤 | 愿意继续刷镜牢的玩家更看重后续游玩资源 |

核心观察指标：

| 指标 | 目的 |
| --- | --- |
| 购买转化率 | 哪一版更容易促成购买 |
| 购买后 7 日镜牢参与次数 | 判断礼包更适合“代练”还是“门票”定位 |
| 购买后通行证等级提升 | 判断玩家是否继续推进 |
| 退款率 / 负反馈 | 判断价值预期是否清晰 |

不建议测试“直接赠送大量碎片箱”的版本，因为它可能引发经济和公平风险。

## 16. 风险与应对

| 风险 | 说明 | 应对 |
| --- | --- | --- |
| 老玩家觉得不公平 | 赛季末礼包价值过高 | 不补狂气、不补纪念、不补 120 级后溢出 |
| 重度玩家被背刺 | 已购玩家通过直升加速进入无限箱 | 已购玩家不卖等级直升 |

| 赛季延期背刺 | 玩家本可自己肝，却已购买直升 | 公告说明 + 小额体力安抚 |

## 17. 首版范围

### 首版必须实现

- 赛季末 28 天上架。
- 一档标准追赶组合包。
- 仅未购付费通行证且低于 Lv.80 的玩家展示。
- 已购通行证玩家只展示不含等级的养成补给包。
- 自动等级直升 20 级。
- 直升上限 Lv.80。
- 超出等级作废，不做补偿。
- 购买后立即自动生效。
- 体力资源 + 经验券 + 纺锤 + 普通材料包。
- 通行证界面 Banner 与详情页。
- 购买后奖励汇总弹窗。
- 基础埋点。

### 首版暂不实现

- 多档复杂礼包。
- 个性化动态定价。
- 大量自选碎片箱补发。
- 赛季纪念奖励补发。
- 已购玩家等级直升。
- 直升券道具化。

暂不实现原因：控制开发量与舆情风险，先验证追赶组合包对赛季末购买率和留存的提升是否成立。

## 18. 总结

赛季末通行证追赶组合包的核心价值，是把“现在买也拿不完”的负面预期，转化为“现在买还能追上关键奖励段，并继续通过日常和镜牢推进”的正向预期。

但追赶方案必须处理好时间价值和公平性：

1. 未购玩家可以通过组合包补票并追赶关键等级。
2. 已购玩家不能购买等级直升，避免加速进入无限箱。
3. 晚买玩家需要支付时间溢价，避免伤害中期通行证销量。
4. 等级直升立即生效并设置上限，避免道具转换和溢出补偿争议。
5. 追赶任务用固定经验，并纳入预计等级展示，帮助玩家判断购买价值。

该方案不应设计成简单高价值礼包，而应服务于《边狱巴士》通行证本身的核心循环：

```text
付费通行证 -> 等级追赶 -> 体力投入 -> 镜牢/日常 -> 通行证经验 -> 碎片箱 -> 人格兑换
```

从商业化角度看，它提高赛季末购买转化；从体验角度看，它降低新玩家错过感；从系统平衡角度看，它通过限制等级上限、控制资源类型、不补无限溢出收益，尽量避免破坏早期玩家与重度玩家的公平感。
